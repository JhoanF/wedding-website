== Description ==

Classic Wedding Theme is perfect for business related to wedding like for wedding planner. You can also use it to showcase your wedding events, wedding gallery, engagement gallery, etc. It can also be used for anniversaries, birthday parties, family functions, catering, boutique flowershop, family gatherings, baby shower, boutiquewedding bands or for art and design, personal, portfolio, girlie or feminine blogs and websites. You can also showcase your romantic journey through this beautiful theme or announce your wedding in an aesthetic manner. The theme has a completely responsive design which looks flawless on all devices. The design of the theme is simple yet attractive and luxurious for people who are ready to tie the knot. It comes with a lot of features for you to create that perfect website. Like featured content and slider, page builder, video display, etc. This theme helps you connect with your guests, as well as share the wedding information and your happy moments. It is optimized for speed to provide an excellent user experience to your website visitors. You can also inform your guests about wedding events, party schedules, and rsvp with this theme in a stylish manner.

== Homepage Installation ==

Step-1. Go to Dashboard -- Pages -- Add new page -- Select (Home Page) in templates -- Publish it.
Step-2. Go to Dashboard -- Settings -- Reading -- Click on static page -- Select home page -- Save changes.

Layout Section
Step-1. Go to Dashboard -- Appearnace -- Customize -- Theme option panel -- Manage Site Layout Section -- Check to Box Layout for box width layout.  -- Publish it.

Header Section
Step-1. Go to Dashboard -- Appearnace -- Customize -- Theme option panel -- Manage Header Section -- Add the button link and text. -- Publish it.

Slider Section
Step-1. Go to Dashboard -- Posts -- Add new Category -- Add new post -- Add title, content, featured image and select category -- Publish it.
Step-2. Go to Dashboard -- Appearnace -- Customize -- Theme option panel -- Manage Slider Section -- Select the category -- Publish it.

About Section
Step-1. Go to Dashboard -- Pages -- Add new page -- Add title, content, featured image -- Publish it.
Step-2. Go to Dashboard -- Appearnace -- Customize -- Theme option panel -- Manage About us Section -- Select the page -- Publish it.

Footer Section
Step-1. Go to Dashboard -- Appearnace -- Customize -- Theme option panel -- Manage Footer Section -- Add copyright text -- Publish it.
